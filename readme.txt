==Available options==

**The following options can be set by defining constants. **

	* __MTF_AVATAR_DEFAULT__ - url to default avatar. If false, will use the wordpress option. 

**The following options can be set by setting a global variable. **

	* __$mtf_featured_image_info__ - Set info/dimensions of the featured image: ( 'x' => n, 'y' => n )
